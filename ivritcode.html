IvritCode Lab

Exploring Hebrew letters as operations of creation — a symbolic operating system of light.

What is IvritCode?

IvritCode is a symbolic language in which each Hebrew letter corresponds to a core operation of creation, and each pair of letters (a Gate) is a higher-order macro instruction.

It is not only about computers. It is a way to think about how the universe moves, remembers, binds, and reveals.

Example (Gate אב)

# Pseudo-IvritCode
אב ;    # Gate of Emergence
        # Spark chooses form; breath enters a house.

In the future, this Lab will host interactive visualizations, simple interpreters, and guided meditations on the Gates as code.
